Portfolio

BetterReader

BetterReader is a Japanese comic book reader, featuring a dynamic and material design inspired UI. A while back, I'd looked through Google's Play Store for an app to read manga -- I found one that works, but knew that I could make it better. So, the main motivation for this app was to create a premium experience, for free. The app also addresses the issues I had with many of the manga readers currently in the Play Store, such as intrusive ad placement, ambiguous UI components, and broken functionality.

OnTime

OnTime is a Boston bus tracker app that lets users set alarms and receive notifications when their bus is leaving. The main motivation for this app was to help users pace their morning routines, circumventing the need to directly look up bus info before school or work. OnTime uses the MBTA API to track buses in real-time and will also give an ETA for a given user's destination.

Clime

Clime is a weather app, and is also one of the first apps that I'm actually proud of developing. I made the app mainly to experience using a good amount of Android's native UI components - such as list/recycler views, making web requests/RESTful API calls asynchronously, requesting for certain user permissions - such as access to internet and location, persisting data locally, and using best practices in general.

CollegeSmart

CollegeSmart is a simple calculator that helps new college grads budget their loan payments over 5, 10, or 15 years. It also includes cost of living simulator for New York, Boston, Chicago, Seattle, and Los Angeles based on your expected income, loan payments, income tax, interest rates, and the relative cost of goods in each city.
